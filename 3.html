Лекция №3

Базовый шаблон страницы

        <!DOCTYPE html>
        <html lang="ru">
        <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1">
            <title>Сайт начинающего фронтендера</title>
        </head>
        <body>
        <!-- Здесь будет содержимое страницы-->
        </body>
        </html>

Тег <main>

      <main>
     Привет, я основной контент!
     Нахожусь только на этой странице.
</main>

Спецификация не допускает использование на одной странице более одного тега <main>, если у них нет специального атрибута hidden

Расшифровка цветовой схемы

  • Пунктиром выделен особый тег <body>;
  • Синей рамкой выделяются поточные теги;
  • Фиолетовой рамкой выделяются теги для создания смысловых разделов;
  • Оранжевой рамкой выделяются заголовочные теги;
  • Розовой рамкой выделяются поточные теги c текстом.
